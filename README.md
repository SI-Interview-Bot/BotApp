# Interview Bot

## Setting up

1. Install `node`:
    - `sudo apt update`
    - `sudo apt install nodejs`
    - `node -v`
2. Install `npm`:
    - `sudo apt install npm`
3. Install `yarn`:
    - `sudo npm install -g yarn`
